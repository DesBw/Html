Evaluating native Text Editors

Whisk is a lightweight editor for HTML, PHP, CSS, and JavaScript development.

Code written in the editor will immediately output to the preview pane. This instant feedback clarifies the effects of your changes making development more efficient. It is great for learning web development, testing snippets of code, or building entire web pages.

Whisk has no autocomplete

I was planning to use Whisk due to recommendations from reddit. Turns out, Whisk cannot autocomplete the tags. So, I looked around for other options.

CodeRunner seems better than Whisk

CodeRunner has both the autocomplete feature, as well as the live preview of the html. For this, it is better than Whisk.

CodeRunner is better than Nova

  • Nova is an expensive and popular text editor in the mac sphere. It has the code autocomplete and html preview. But, I couldn't find the way to run JavaScript within the editor. That is very strange. It seems to support JavaScript; but running the code seems impossible without going through some kind of hassle.
  • Another advantage of the CodeRunner is I can choose node.js as well as browser version within the app to run JavaScript code. So, I can use both console.log and alert functions
